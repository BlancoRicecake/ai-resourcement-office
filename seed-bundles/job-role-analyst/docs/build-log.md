# Build Log

## Source Problem

AI력 사무소는 실제 직무처럼 작동하는 에이전트를 만들기 위해, 제작 전 단계에서 역할과 업무 구조를 정확히 해부할 직원이 필요하다.

## MVP Scope

사용자의 에이전트 아이디어를 받아 실제 직무 매핑, 업무 범위, 업무 흐름, 사용자 컨펌 지점, 협업 직무, 추가 에이전트 후보를 정리하는 Markdown 설계서를 만든다.

## Next Improvements

- 직무별 설계서 템플릿 축적
- 산업군별 질문 세트 추가
- 에이전트 제작 번들과의 후속 연결
- 조사 도구를 연결한 최신 직무/시장 검증 단계 추가
