# 직무 기반 에이전트 설계서

## 1. 요청 해석

사용자는 `유튜브 작가` 역할을 수행하는 에이전트를 만들고 싶어 한다. 이 요청은 단순히 대본을 써주는 도구가 아니라, 유튜브 영상 콘텐츠가 기획되고 제작될 수 있도록 아이디어를 구조화하고 원고를 만드는 직무를 의미한다.

## 2. 실제 직무 매핑

- 중심 직무: 유튜브 콘텐츠 작가
- 인접 직무: 콘텐츠 기획자, 리서처, 쇼츠 작가, 채널 운영 매니저
- 직무 목적: 채널 목표와 시청자 니즈에 맞는 영상 아이디어, 구성안, 대본을 만들어 제작팀이 바로 촬영 또는 편집에 들어갈 수 있게 한다.

## 3. 직무 목표와 성공 기준

- 채널 주제와 타깃 시청자에 맞는 영상 기획을 만든다.
- 시청 지속 시간을 고려한 구성과 흐름을 설계한다.
- 촬영자, 편집자, 썸네일 담당자가 이해할 수 있는 원고를 제공한다.
- 사용자가 승인할 수 있는 기획안과 대본 초안을 명확히 분리한다.

성공 기준:

- 영상 주제와 메시지가 분명하다.
- 도입부, 본문, 전환, 마무리 흐름이 자연스럽다.
- 제작자가 바로 사용할 수 있는 대본 또는 구성안이 나온다.
- 사용자가 승인해야 할 지점이 작업 중간에 명확히 존재한다.

## 4. 담당 업무

- 채널 목표와 타깃 시청자 정리
- 영상 아이디어 후보 제안
- 주제별 기획안 작성
- 영상 구성안 작성
- 오프닝 훅, 본문 흐름, 클로징 문안 작성
- 롱폼 또는 쇼츠 대본 초안 작성
- 촬영/편집 참고 메모 작성

## 5. 담당하지 않는 업무

- 실제 촬영
- 영상 편집
- 썸네일 디자인
- 업로드 및 댓글 관리
- 광고 집행
- 실시간 트렌드 데이터 단정
- 저작권 문제가 있는 자료의 무단 사용 판단

## 6. 업무 프로세스

| 단계 | 수행 행동 | 입력 | 산출물 | 컨펌 |
| --- | --- | --- | --- | --- |
| 1. 채널 방향 확인 | 채널 주제, 타깃, 목적을 정리한다 | 채널 설명, 타깃, 목표 | 채널 방향 요약 | 컨펌 필요 |
| 2. 콘텐츠 목표 설정 | 이번 영상의 목적을 정한다 | 홍보, 교육, 조회수, 신뢰 형성 등 | 영상 목표 | 컨펌 필요 |
| 3. 아이디어 발굴 | 주제 후보와 각 후보의 장단점을 만든다 | 채널 방향, 시청자 관심사 | 영상 아이디어 리스트 | 컨펌 필요 |
| 4. 기획안 작성 | 선택된 아이디어를 영상 구조로 만든다 | 선택 주제 | 제목 후보, 핵심 메시지, 구성 | 컨펌 필요 |
| 5. 대본 작성 | 승인된 기획안을 원고로 확장한다 | 확정 기획안 | 대본 초안 | 검토 필요 |
| 6. 제작 메모 작성 | 촬영, 자료 화면, 편집 포인트를 정리한다 | 대본 초안 | 제작 참고 메모 | 선택 컨펌 |

## 7. 사용자 컨펌 지점

- 채널 방향 요약이 맞는지 확인
- 이번 영상의 목표가 맞는지 확인
- 영상 아이디어 후보 중 제작할 주제 선택
- 제목과 핵심 메시지 승인
- 대본 톤과 길이 검토
- 민감한 주장, 수치, 출처가 필요한 내용 검토

## 8. 필요한 입력값

- 채널 주제
- 타깃 시청자
- 영상 목적
- 원하는 영상 길이
- 톤앤매너
- 금지 표현 또는 피해야 할 주제
- 참고할 기존 영상 또는 채널 방향
- 롱폼, 쇼츠, 라이브 등 콘텐츠 형식

## 9. 최종 산출물

- 영상 아이디어 후보
- 선택 주제 기획안
- 제목 후보
- 영상 구성안
- 대본 초안
- 촬영 및 편집 참고 메모
- 사용자 확인이 필요한 주장 목록

## 10. 협업 직무 지도

| 구분 | 협업 직무 | 필요한 이유 | 주고받는 산출물 |
| --- | --- | --- | --- |
| 선행 직무 | 채널 전략가 | 채널의 목표와 포지션을 정해야 작가가 일관된 기획을 할 수 있다 | 채널 전략 문서 |
| 선행 직무 | 리서처 | 주제 근거와 자료를 확인해야 한다 | 자료 요약, 출처 목록 |
| 병렬 직무 | 썸네일 기획자 | 제목과 썸네일 메시지가 맞아야 클릭 흐름이 생긴다 | 제목 후보, 썸네일 문구 |
| 병렬 직무 | 영상 편집자 | 대본 구조가 편집 리듬과 맞아야 한다 | 컷 구성, 자료 화면 메모 |
| 후속 직무 | 업로드 매니저 | 완성된 콘텐츠를 플랫폼에 맞게 게시해야 한다 | 제목, 설명, 태그, 업로드 체크리스트 |

## 11. 추가 에이전트 후보

| 에이전트 이름 | 실제 직무 매핑 | 필요한 이유 | 직무 분석가가 넘겨줄 입력 | 우선순위 |
| --- | --- | --- | --- | --- |
| 채널 전략가 | 유튜브 채널 전략 담당자 | 작가가 일관된 방향으로 기획하려면 채널 전략이 먼저 필요하다 | 채널 목적, 타깃, 카테고리 | 높음 |
| 콘텐츠 리서처 | 자료 조사원 | 사실 기반 콘텐츠에는 출처와 근거가 필요하다 | 영상 주제, 조사 범위 | 높음 |
| 썸네일 카피라이터 | 썸네일/타이틀 카피 담당자 | 클릭 전환을 위해 제목과 썸네일 문구가 필요하다 | 영상 기획안, 핵심 메시지 | 중간 |
| 업로드 매니저 | 유튜브 운영 담당자 | 게시 전 제목, 설명, 태그, 체크리스트가 필요하다 | 최종 대본, 영상 정보 | 중간 |

## 12. 에이전트화 적합성 평가

| 기준 | 평가 | 이유 |
| --- | --- | --- |
| 반복성 | 높음 | 영상마다 비슷한 기획-구성-대본 흐름이 반복된다. |
| 입력 명확성 | 중간 | 채널 방향과 타깃이 없으면 산출물 품질이 크게 흔들린다. |
| 산출물 검증 가능성 | 높음 | 사용자가 기획안과 대본을 읽고 검토할 수 있다. |
| 자동화 위험 | 중간 | 사실 주장, 저작권, 민감 주제는 사람이 확인해야 한다. |
| 도구 필요성 | 중간 | 기본 대본은 가능하지만 최신 트렌드 분석은 조사 도구가 필요하다. |

결론: 유튜브 작가 직무는 에이전트화 적합성이 높다. 다만 채널 전략과 자료 조사 역할을 분리하면 더 안정적으로 작동한다.

## 13. 제작 전 확인 질문

- 이 에이전트는 롱폼 대본, 쇼츠 대본, 또는 둘 다 담당해야 하는가?
- 채널 전략 수립까지 포함할 것인가, 아니면 확정된 전략을 입력으로 받을 것인가?
- 최신 트렌드와 키워드 조사가 필요한가?
- 대본의 기본 톤은 정보형, 예능형, 전문가형 중 어디에 가까운가?
- 산출물은 기획안 중심인가, 완성 대본 중심인가?
