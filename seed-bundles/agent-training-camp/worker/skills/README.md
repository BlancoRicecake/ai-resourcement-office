# skills

에이전트 훈련소가 훈련 과정에서 따르는 규칙 모음이다.

- `legal-ingestion-rules.md`: 합법적 지식 수집 규칙 (아이디어/표현 이분법, 허용 소스, 금지 행위)
- `synthesis-to-practice-rules.md`: 학술 개념을 실무 지침으로 번역하는 규칙 (핵심 단계)
- `memory-loading-rules.md`: 번역한 지침을 신임 에이전트 메모리에 적재하는 규칙
- `source-credibility-rules.md`: 소스 공신력·최신성·합법성 판단 규칙
- `graduation-exam-rules.md`: 직무 맞춤 졸업 시험 생성·채점·불합격 처방 규칙

이 규칙들은 `worker/agent.md`의 4단계 파이프라인(흡수 → 합성 → 적재 → 졸업 시험)을 구체화한다.
