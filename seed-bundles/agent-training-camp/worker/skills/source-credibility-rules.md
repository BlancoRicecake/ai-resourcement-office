# Source Credibility Rules — 소스 공신력 판단 규칙

훈련소는 아무 자료나 학습시키지 않는다. 논문, 학술자료, 저명 학자의 저서·강연처럼 공신력 있는 정보만 실무 지침의 근거로 삼는다.

## 필터링 기준

| 기준 | 질문 |
| --- | --- |
| 공신력 | 저자·기관·저널이 믿을 만한가? 동료평가(peer review)를 거쳤는가? |
| 최신성 | 이 지식은 지금도 유효한가? 플랫폼/시장/규제가 바뀌지 않았는가? |
| 재현성 | 방법론이 재현 가능하고, 결과가 다른 연구와 일관되는가? |
| 사업 적합성 | 대상 산업·비즈니스 모델·타깃에 실제로 맞는가? |
| 합법성 | 오픈 액세스 또는 TDM 허용 소스인가? |

## 소스 등급

- **A (원리 근거)**: 동료평가 논문, 표준 문서, 저명 학자의 검증된 이론. 실무 지침의 기반으로 사용.
- **B (실무 보정)**: 공신력 기관 리포트, 산업 벤치마크. 최신 시장 실행안 보정에 사용.
- **C (검증 후보)**: 블로그, 사례담, 단일 출처 주장. 실무 지침으로 확정하지 않고 `RESEARCH.md`에 검증 후보로만 둔다.

## 적용 규칙

- 오래된 학술 이론은 기본 원리(A급)로 쓸 수 있지만, 시장 실행안은 최신 자료(B급)로 보정한다.
- 기발하지만 근거가 약한 주장은 실무 지침이 아니라 아이디어 후보로만 둔다.
- 단일 출처 주장은 교차 확인 전까지 확정하지 않는다.
- 고위험 직무(법률·의료·금융·투자·채용)는 A급 근거라도 전문가 검토·규제 확인 문구를 함께 남긴다.

## 출처 표기 형식

```txt
- 저자(연도). 제목. 출처/저널. URL. [등급 A/B/C]
```

원문 본문은 저장하지 않는다. 위 한 줄 표기만 `RESEARCH.md`에 남긴다.
