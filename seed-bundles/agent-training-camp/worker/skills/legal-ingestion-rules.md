# Legal Ingestion Rules — 합법적 지식 수집 규칙

에이전트 훈련소는 '크롤러'가 아니라 '사내 지식 번역가'다. 원문을 복사·저장하지 않고 아이디어만 추출한다.

## 저작권의 핵심: 아이디어와 표현의 이분법

- 저작권은 작가의 **표현(문장 그 자체)**을 보호한다.
- 저작권은 그 안의 **아이디어(이론, 공식, 사실, 방법론)**를 보호하지 않는다.
- 따라서 원문을 그대로 저장하지 말고, 개념·팩트만 추출해 **자기 언어로 다시 요약**한다(변형적 이용).
- 이렇게 하면 법적 리스크가 극적으로 낮아지고, 신임 에이전트가 읽을 지침의 노이즈도 준다.

## 허용 소스 (우선순위)

| 소스 | 성격 | 비고 |
| --- | --- | --- |
| arXiv | CS/수학/물리 프리프린트 | 초록·본문 공개 |
| Semantic Scholar API | 논문 메타데이터/초록 | TDM 친화적 |
| OpenAlex | 학술 메타데이터 그래프 | 개방 API |
| PubMed | 의학/생명과학 초록 | 초록 공개 |
| DOAJ | 오픈 액세스 저널 디렉토리 | 전문 공개 다수 |
| 공신력 기관 리포트 | 정부·표준기구·대학 | 라이선스 확인 |
| 저명 학자의 공개 강연·초록 | 공개 발표 | 인용 표기 |

## 금지 행위

- 유료 논문·전자책 원문을 크롤링해 전문 저장하기.
- 페이월·비공개 자료를 우회 수집하기.
- TDM(텍스트·데이터 마이닝)을 금지한 사이트를 자동 수집하기.
- 원문 문장·긴 인용문을 신임 에이전트의 메모리에 그대로 넣기.

## 추출 규칙

1. 초록(Abstract)과 결론(Conclusion)을 우선 읽는다 — 핵심 논리와 결과가 압축돼 있다.
2. 방법론·프레임워크·공식·핵심 팩트만 뽑는다.
3. 원문 문장 대신 개념을 자기 문장으로 다시 쓴다.
4. 인용이 꼭 필요하면 짧은 출처 표기(저자·연도·제목·URL)만 남긴다.
5. 원문 텍스트는 훈련이 끝나면 버린다. 메모리에는 남기지 않는다.

## 검색 불가 환경 대응

- 온라인 검색·API가 없으면, 이미 검증된 기본 이론(원리 수준)만 사용한다.
- 이때 모든 항목에 "최신 자료 재확인 필요"를 표시한다.
- 최신 시장·플랫폼·가격·규제 정보는 추정하지 말고 "조사 필요"로 남긴다.
