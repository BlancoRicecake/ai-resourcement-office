# Security Notes

- 신임 에이전트의 메모리 파일에 API 키, 비밀번호, 토큰, 개인정보를 저장하지 않는다.
- 원문 발췌·긴 인용문을 메모리에 저장하지 않는다. 새로 쓴 실무 지침과 짧은 출처 표기만 남긴다.
- 유료·비공개 자료를 우회 수집하지 않는다. 오픈 액세스·TDM 허용 소스만 사용한다.
- 기존 메모리 파일을 덮어쓰기 전에 사용자 의도를 확인한다.
- 외부 검색·API 호출·파일 쓰기는 사용자 승인과 실행 환경 권한 범위 안에서만 수행한다.
- 고위험 직무(법률·의료·금융·투자·채용) 지식에는 전문가 검토·규제 확인 문구를 함께 적재한다.

## 저작권 안전 요약

저작권은 표현(문장)을 보호하고 아이디어(이론·공식·사실·방법론)는 보호하지 않는다. 훈련소는 원문을 저장하지 않고 아이디어만 추출해 자기 언어로 다시 씀으로써(변형적 이용) 저작권 침해 리스크를 낮춘다. 다만 최종 법적 판단과 자료 사용 범위는 사용자 책임이다.
