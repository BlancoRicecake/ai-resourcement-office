# Limitations

- 이 번들은 호스팅 서비스가 아니다. 사용자의 모델 환경에서 직접 실행한다.
- 훈련소는 모델을 파인튜닝하지 않는다. `memory/` 파일에 지식을 적재할 뿐, 신임 에이전트가 이를 매 세션 다시 읽어야 효과가 난다.
- 온라인 검색·API가 없으면 최신 시장·플랫폼·가격·규제 정보를 확보할 수 없어 기본 원리 수준으로만 훈련한다. 이때 항목에 "조사 필요"를 표시한다.
- 적재된 지식은 훈련 시점 기준이다. 시간이 지나면 재확인·갱신이 필요하다.
- 저작권·라이선스의 최종 판단, 고위험 직무 지식의 전문가 검토, 메모리 적재 확정은 사용자 책임이다.
- 신임 에이전트가 실제 실행 권한이나 완벽한 최신 지식을 가진 것처럼 과장하지 않는다.
- 실무 번역의 품질은 소스 공신력과 대상 산업 정보의 구체성에 좌우된다. 입력이 모호하면 지침도 일반론에 그친다.
