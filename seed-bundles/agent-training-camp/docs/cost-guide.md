# Cost Guide

- 이 번들은 필수 API 키가 없다. 사용자의 모델 계정(Claude/Codex/기타)에서 실행한다.
- 비용은 사용자가 선택한 모델·플랜과 훈련 분량(읽는 소스 수, 번역 지식 항목 수)에 비례한다.
- 온라인 리서치 없이 기본 원리 수준으로만 훈련하면 추가 비용이 거의 없다.
- 선택적으로 학술 검색 API(예: Semantic Scholar)를 붙이면 해당 API의 요금·레이트리밋이 적용된다. 대부분 오픈 액세스 메타데이터/초록은 무료 또는 저비용이다.
- 유료 논문 전문 구매·크롤링은 하지 않으므로 콘텐츠 구매 비용이 발생하지 않는다.

비용을 줄이려면: 커리큘럼 주제를 핵심 역량으로 좁히고, 라운드를 나눠 필요한 주제부터 훈련한다.
