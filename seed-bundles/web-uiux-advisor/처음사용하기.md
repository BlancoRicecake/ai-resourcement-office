# UI·UX 자문가 처음 사용하기

1. ZIP 압축을 풉니다.
2. Codex에서는 `폴더 열기`로 이 폴더를 선택합니다. Claude Code에서는 이
   폴더에서 터미널을 열고 `claude`를 입력합니다.
3. 첫 메시지로 아래 문장을 그대로 입력합니다.

> 이 폴더의 설명서를 읽고 UI·UX 자문가로 시작해줘. 검토할 서비스와 사용자를
> 한 번에 하나씩 쉽게 물어봐줘.

4. 사이트 주소, 스크린샷, HTML, 기획안 중 편한 자료 하나를 제공합니다.
5. `이 화면에서 사용자가 막힐 부분을 찾아 중요한 순서대로 쉽게 설명해줘`라고
   요청합니다.

Figma나 별도 디자인 도구를 연결하지 않아도 스크린샷만으로 시작할 수 있습니다.
