# Cost Guide

- 이 번들은 필수 API 키가 없다. 하지만 **결정론 도구가 $0으로 처리해 주는 계산이 없다** — 파싱·채점·진단·개선안 작성이 전부 호스트 모델의 판단이므로, 비용은 **전부 호스트 토큰**으로 발생한다.
- 비용은 사용자가 선택한 모델·플랜과 리뷰 분량(화면 수, 순회 횟수, Finding 수, 리포트 길이)에 비례한다.
- **스크린샷 해석 비용**: 화면을 이미지로 넣으면 비전 토큰이 추가로 든다(이미지 입력은 텍스트보다 토큰 단가가 높은 경우가 많다). 대량 스크린샷·고해상도일수록 비용이 커진다. 텍스트 묘사나 렌더된 HTML로 대체하면 비전 비용을 줄일 수 있다.
- **URL fetch 비용**: 호스트 런타임이 페이지를 가져와 컨텍스트에 넣으면 그 HTML/텍스트 분량만큼 입력 토큰이 든다. 긴 페이지는 핵심 섹션만 발췌해 넣는 것이 저렴하다.
- 지식 노드는 INDEX만 먼저 읽고 필요한 노드만 여는 구조라, 전부-로드 대비 토큰을 아낀다.

비용을 줄이려면: (1) 스크린샷 대신 텍스트 묘사·HTML 사용, (2) 리뷰 범위를 특정 플로우로 좁히기, (3) 관련 지식 노드만 열기(INDEX 활용), (4) 리포트를 상위 심각도 중심으로 요청하기.
