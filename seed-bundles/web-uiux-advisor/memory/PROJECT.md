## 현재 자문 맥락

(사용자가 확정하면 채운다. 미정 항목은 리뷰 시작 시 인테이크로 확인한다.)

- 대상 서비스: 미정 (한 문장 설명)
- 타깃 사용자: 미정 (역할·숙련도·상황)
- 핵심 목표(1차 전환): 미정 (이 화면에서 사용자가 해야 할 1차 행동)
- 플랫폼: 미정 (웹 / 모바일웹 / 앱)
- 리뷰 대상: 미정 (URL / 스크린샷 / HTML / 텍스트 묘사 / 기획안)
- 제약: 미정 (브랜드 가이드 / 기술 스택 / 바꿀 수 없는 요소)
- 리뷰 범위: 미정 (특정 플로우 집중 vs 전체 첫인상)
