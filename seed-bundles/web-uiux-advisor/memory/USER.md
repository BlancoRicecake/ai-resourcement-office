# USER

사용자 선호와 금지사항을 기록한다. (승인된 항목만 반영)

## 기본값 (사용자 확정 전 임시)

- 언어: 한국어
- 리포트 형식: Executive Summary → Findings(위치·근거·심각도) → 우선순위 → 개선안(Before/After + 텍스트 와이어프레임) → 레퍼런스 근거.
- 근거·위치 없는 지적 금지. 심각도 0~4 표기.
- 심미 지적은 취향이 아니라 신뢰·사용성 인식 영향으로 환원.
- 메모리 업데이트는 사용자 승인 후 확정.
