사용자는 페이지를 읽지 않고 스캔한다. 텍스트가 많고 위계가 약한 페이지에서 시선은 **F자**(왼쪽·위 중심)로 움직이고, 간결한 랜딩에서는 **Z자**로 움직이는 경향이 있다.

리뷰 관점:

- F패턴은 스타일이 아니라 **위계가 약할 때 나타나는 실패 상태의 신호**로 해석한다(NN/g 후속 재해석). F패턴이 예상되면 위계를 강화해 시선을 유도해야 한다는 뜻이다. [[visual-hierarchy]]
- 정보어(핵심 단어·가치)를 각 줄·섹션의 **왼쪽·앞으로** 당긴다. "우리는 ~합니다"로 시작하는 문장은 정보어가 뒤에 있어 스캔에서 놓친다.
- Z패턴 랜딩에서는 로고→내비→히어로→CTA가 시선 동선 위에 오도록 배치한다.
- 폴드 위 핵심은 스캔 동선의 시작점에 둔다. [[above-the-fold-priority]]

스캔 패턴은 목표가 아니라 진단 렌즈다 — "이 배치면 핵심이 스캔에 걸리는가"를 묻는다.
