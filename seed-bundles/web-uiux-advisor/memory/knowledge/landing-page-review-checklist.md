랜딩페이지를 섹션별로 진단하는 체크리스트다. 제작용 랜딩 프레임을 **평가 프레임으로 역변환**했다. 자세한 절차는 스킬 `landing-page-review.md`.

**Above-fold 5요소** — 스크롤 없이 5초 안에 회상돼야 한다. 빠진 것이 최우선 미스. [[above-the-fold-priority]]
1. 가치제안 헤드라인 / 2. 서브헤드 / 3. 단일 1차 CTA / 4. 제품 비주얼·데모 / 5. 신뢰 신호.

**Mid 섹션** — 혜택 중심 설명, 오브젝션 해소, 사회적 증거를 결정 지점 근처에, 스캔 가능한 위계.

**Bottom 섹션** — 재확인 CTA, FAQ(위장한 반론 목록), 푸터 신뢰 요소.

**공통 실패 패턴**:

- **CTA 과다·경쟁** — 폴드 위 1차 CTA가 여럿. 무엇을 눌러야 할지 모른다.
- **모호한 가치제안** — 헤드라인이 슬로건·자화자찬.
- **증거 하단 매몰** — 후기·수치가 폴드에서 멀어 신뢰가 결정 지점에 없다.
- **낮은 대비·약한 위계 / hover 의존 / 빈 상태 방치**.

폼이 있으면 폼 UX를 함께 본다. [[form-ux-best-practices]] · 전체 절차는 [[heuristic-evaluation-process]].
