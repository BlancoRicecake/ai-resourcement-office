기획 자문·개선안에서 "어떤 시각 방향"을 말할 때 쓰는 스타일 계보 어휘다. MengTo/Skills의 web-design 카탈로그(60여 종)를 계열로 재구조화했다. 자문가는 방향을 **지목**하고, 구현은 넘긴다.

계열 어휘:

- **에디토리얼-테크** — 격자·라인·인덱스 기반 잡지형 레이아웃(framed-grid, editorial-tech, split-layout-technical 계열). 정보 밀도 높은 SaaS·문서에.
- **스큐어모픽/하이컨트라스트-클린** — 입체·질감 강조(skeuomorphic, high-contrast-skeuomorphic). 촉각적 신뢰감.
- **메시 그라디언트/글래스** — 부드러운 그라디언트·유리 표면(mesh-gradient, glass-dark). 감성·프리미엄. 대비 주의 [[glass-dark-ui-pattern]].
- **미니멀 페이퍼/라이트** — 종이 질감·밝은 배경(clean-minimal-beige, light-mode-paper). 가독성·차분함.
- **테크-그리드/블루프린트** — 격자·레이저·WebGL 배경(background-grid, technical-wireframe). 기술 브랜드 미학(트렌드성, 시효 있음).

리뷰 관점:

- 스타일 선택은 **타깃·브랜드·목적**에 종속된다 — "예뻐서"가 아니라 "이 사용자·목적에 맞아서" 고른다.
- 트렌드 계열(테크-그리드 등)은 시효가 있으니 인용 시 시점을 명시한다.
- 실제 사례는 갤러리에서, 규범은 디자인 시스템에서 보강한다. [[reference-galleries]] · [[design-systems-authority]]
