사람은 경험 전체의 평균이 아니라 **절정(peak)과 마지막(end) 순간**으로 경험을 기억한다. 리소스를 균등하게 쓰지 말고 결정적 순간에 몰아 쓴다.

리뷰 관점:

- 온보딩의 "첫 성공(aha)" 순간, 결제/가입 완료 화면, 에러에서 회복되는 순간에 공을 들였는가.
- 완료 화면이 무미건조한 "성공했습니다"에 그치면 기회 손실 — 다음 행동·축하·가치 재확인으로 마무리한다.
- 최악의 순간(에러·실패·대기)이 곧 기억되는 peak일 수 있으니, 마찰 지점의 카피·안내를 특히 본다. [[microcopy-guidelines]]
- 빈 상태·최초 진입은 첫인상 peak이다. [[empty-state-onboarding]]

어디에 리소스를 몰지 결정할 때 이 규칙으로 우선순위를 잡는다.
