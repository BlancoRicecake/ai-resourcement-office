타깃까지의 시간은 거리에 비례하고 크기에 반비례한다. 즉 **크고 가까운 것은 빠르게, 작고 먼 것은 느리게** 눌린다. **[H4]**

리뷰 관점:

- 1차 CTA는 충분히 크고 손이 닿는 위치(모바일은 하단, [[thumb-zone-mobile]])에 둔다. 작은 텍스트 링크만 CTA면 미스.
- 파괴적 액션(삭제·탈퇴·취소)은 일부러 작게·멀리 둬 오조작을 막는다. 확인 CTA와 취소가 같은 크기로 붙어 있으면 미스.
- 인접 타깃 간 간격이 좁으면(특히 터치) 오조작이 는다.
- 화면 모서리·가장자리는 "무한 크기" 타깃처럼 작동한다(포인터가 멈추므로). 데스크톱 주요 액션 배치에 활용.

CTA·타깃의 크기·거리는 시각 위계와 함께 판단한다. [[visual-hierarchy]]
