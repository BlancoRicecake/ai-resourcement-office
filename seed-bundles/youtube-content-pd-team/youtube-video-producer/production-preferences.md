# Production Preferences

사용자가 컨펌한 반복 제작 규칙만 저장한다.

## Rule Format

```md
- Rule:
- Applies to:
- Example:
- Confirmed by user:
- Date:
```

## Examples

```md
- Rule: 자막은 한 줄 14자 이내로 짧게 유지한다.
- Applies to: Shorts, product demos
- Example: "시간을 줄이는 방법"
- Confirmed by user: yes
- Date: YYYY-MM-DD
```
