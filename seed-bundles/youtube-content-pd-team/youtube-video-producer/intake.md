# Intake: 영상 제작자 선행 질문

## 첫 질문

이 영상을 어떤 제작 형식으로 만들고 싶나요?

## 선택지

- PPT/슬라이드형: 무료 또는 로컬 제작에 가장 적합합니다.
- 이미지 카드형: 이미지 생성 API가 필요할 수 있습니다.
- 화면 녹화형: 사용자의 제품, 웹사이트, 앱을 보여줄 때 적합합니다.
- 실사 촬영형: 촬영표와 편집 가이드를 제공합니다.
- AI 영상 생성형: 영상 생성 API 비용이 발생할 수 있습니다.
- TTS/더빙형: 음성 API 비용이 발생할 수 있습니다.
- 아직 미정: 예산과 목적에 맞춰 추천받고 싶습니다.

## 두 번째 질문

예산과 외부 API 사용 의향을 확인한다.

- 무료/로컬 제작만 원함
- 낮은 비용이면 외부 API 사용 가능
- 품질이 중요하면 유료 API 사용 가능
- 아직 모르니 비용 설명을 먼저 듣고 싶음

## 세 번째 질문

선호 스타일을 확인한다.

- 미니멀하고 깔끔한 스타일
- 역동적이고 빠른 편집
- 정보 밀도가 높은 교육형
- 브랜드 감성이 강한 고급형
- 직접 예시를 제공하고 싶음
