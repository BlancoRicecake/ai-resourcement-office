# Revision Rules: 제작 피드백 반영

영상 제작 피드백은 양이 많을 수 있으므로 위치와 방향성을 분리해 처리한다.

## 바로 반영 가능한 피드백

- 특정 장면 삭제
- 특정 자막 수정
- B-roll 교체
- 썸네일 문구 수정
- 컷 길이 조정

## 선택지가 필요한 피드백

다음처럼 추상적인 피드백은 바로 수정하지 말고 선택지를 제시한다.

- 더 세련되게
- 더 빠르게
- 더 쉽게
- 더 고급스럽게
- 더 유튜브스럽게
- 더 판매되게

## 방향성 선택지 예시

- 속도 중심: 컷을 짧게, 자막을 강하게, 장면 전환을 빠르게
- 신뢰 중심: 자료 화면, 근거 자막, 차분한 전환
- 브랜드 중심: 색감 통일, 여백, 고급스러운 썸네일
- 전환 중심: 문제-해결-증거-CTA 구조 강화

## 컨펌 질문

수정 후 다음처럼 묻는다.

> 이 방향을 앞으로도 기본 제작 스타일로 저장할까요?

사용자가 동의한 경우에만 `production-preferences.md`에 저장한다.
