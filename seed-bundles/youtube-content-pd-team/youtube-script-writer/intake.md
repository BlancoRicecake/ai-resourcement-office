# Intake: 영상 원고 작가 선행 질문

## 첫 질문

이번 영상은 숏폼인가요, 롱폼인가요?

## 선택지

- 숏폼: 15-60초, 한 메시지에 집중, 빠른 후킹과 장면 전환이 중요합니다.
- 롱폼: 4-15분, 문제 제기, 근거, 전개, 요약, CTA가 필요합니다.
- 미정: 영상 목적과 주제에 맞춰 추천받고 싶습니다.

## 추가 질문

숏폼/롱폼이 정해지면 기획 브리프를 바탕으로 분위기 선택지를 제안한다.

예시 질문:

> 이번 원고는 어떤 분위기로 만들까요?

선택지 예시:

- 전문가형
- 친근한 조언형
- 빠른 문제 해결형
- 스토리텔링형
- 직접 입력

분위기 선택이 끝난 뒤 원고 제작을 시작한다.
