# Workflow: 채널 운영과 성과 회고

1. 영상 제목, 설명란, 챕터, 고정 댓글을 만든다.
2. 썸네일/제목 실험 후보를 정리한다.
3. 게시 후 `data-access.md` 기준으로 데이터 제공 방식을 확인하고
   YouTube Studio 지표를 받는다.
4. CTR, 평균 시청 지속 시간, 이탈 구간을 해석한다.
5. 채널 최근 영상들과 비교해 상위 성과 영상을 식별하고,
   `performance-analysis.md` 절차로 차별점과 근거를 정리한다.
6. 다음 영상의 후킹, 구조, 썸네일, CTA 개선안을 출력한다.
7. 개선안 중 이번 사이클에 반영할 핵심 변경 1-3개를 골라
   `handoff-to-team.md` 형식으로 기획자, 원고 작가, 영상 제작자에게 전달한다.
