# CHANNEL

팀이 함께 운영하는 유튜브 채널의 맥락을 저장하는 파일이다. 팀원 4명이 공유한다.

## 채널 정체성

- 채널명/URL: 미정
- 채널 목적: 미정
- 기존 채널 / 신규 채널 여부: 미정

## 분야와 포지셔닝

- 분야: 미정
- 차별화 포인트: 미정
- 경쟁/레퍼런스 채널: 미정

## 타깃 시청자

- 타깃 시청자: 미정
- 시청자 문제: 미정
- 영상 약속: 미정

## 톤과 형식

- 원고 분위기: 미정
- 영상 스타일: 미정
- 숏폼/롱폼 기본값: 미정

## 운영 조건

- 업로드 주기: 미정
- 운영 가능 자원: 미정
- 외부 API 사용 가능 여부와 예산 기준: 미정
- 데이터 제공 방식(수동 입력/CSV/API): 미정

## 비즈니스 맥락

- 제품, 서비스, 브랜드 맥락: 미정
- 채널의 비즈니스 목적: 미정

## 업데이트 규칙

채널 정보가 확인되면 바로 확정하지 말고 `CHANNEL.md 후보`로 제안한다.
