# PRODUCTION-LOG

제작 사이클별 기록을 누적하는 파일이다. 기획 → 원고 → 제작 → 업로드 → 성과 회고로 이어지는 한 사이클이 끝날 때마다 채널 매니저의 분석과 각 팀원이 배운 점을 기록해, 다음 사이클의 입력으로 쓴다.

## 기록 원칙

- 성과 지표는 데이터 출처(수동 입력/CSV/API)를 함께 적는다.
- 지표로 뒷받침되지 않는 판단은 `[가설]`로 표시한다.
- 개선 지시는 근거 지표와 검증 지표를 함께 적고, 사이클당 1-3개로 제한한다. 나머지 발견은 `[대기]`로 남긴다.
- 기록은 자동 확정하지 않고 사용자 승인 후 반영한다.

## 기록 형식

```md
## YYYY-MM-DD 사이클 N: 영상 제목/주제

- 형식: 숏폼/롱폼
- 기획 요약:
- 성과 지표 (데이터 출처: 수동 입력/CSV/API):
  - CTR:
  - 평균 시청 지속 시간:
  - 이탈 구간:
  - 채널 평균 대비:
- 채널 매니저 핵심 발견 (1-3개):
- 개선 지시:
  - 기획자에게: (지시 / 근거 / 검증 지표)
  - 원고 작가에게: (지시 / 근거 / 검증 지표)
  - 제작자에게: (지시 / 근거 / 검증 지표)
- 대기 항목 `[대기]`:
- 각 팀원이 배운 점:
  - 기획자:
  - 원고 작가:
  - 제작자:
  - 채널 매니저:
- 다음 사이클 반영 사항:
```
