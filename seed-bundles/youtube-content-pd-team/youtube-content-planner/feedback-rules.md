# Feedback Rules

사용자 피드백은 자동으로 영구 규칙이 되지 않는다.

## 저장 절차

1. 사용자의 피드백을 요약한다.
2. 다음 질문으로 확인한다:

   > 앞으로도 이 채널에서는 이 기준을 계속 적용할까요?

3. 사용자가 동의하면 아래 형식으로 저장한다.

## Rule Format

```md
- Rule:
- Reason:
- Applies to:
- Confirmed by user:
- Date:
```

## 예시

```md
- Rule: 과장된 수익 표현을 쓰지 않는다.
- Reason: 채널 신뢰도를 유지하기 위해서.
- Applies to: 제목, 후킹, CTA, 썸네일 문구
- Confirmed by user: yes
- Date: YYYY-MM-DD
```
