진단은 언제나 above-the-fold부터 본다. 방문자는 5초 안에 "무엇을·누구를 위해·다음 행동"을
회상할 수 있어야 한다. 히어로 영역에서 다음 5요소를 인벤토리한다. **[H2]**

1. 가치제안 헤드라인 — 무엇을 파는지 한 문장으로. 자세히는 [[headline-clarity]].
2. 서브헤드 — 헤드라인을 보강하는 한 겹.
3. 단일 CTA — 목표 행동 하나. 링크 과잉은 [[attention-ratio]] 미스로 이어진다.
4. 제품 비주얼 — 무엇인지 즉시 보이게.
5. 신뢰 신호 — 후기·로고·수치. 배치는 [[social-proof-placement]] 원칙을 따른다.

빠진 요소가 있으면 그것이 **최우선 미스**다.
