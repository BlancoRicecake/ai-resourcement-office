카피가 잘 안 읽히면 **F-패턴** 스캔을 의심한다. 방문자는 페이지를 정독하지 않고 왼쪽 위에서
가로로, 다시 아래로 훑는다. 따라서 문단·헤드라인·불릿의 **왼쪽과 앞머리에 정보어**(핵심 명사·
결과)를 배치해야 스캔 중에도 메시지가 전달된다. **[H5]**

수식어로 시작하는 문장, 결론이 뒤에 숨은 문단은 스캔에서 탈락한다. 앞머리를 정보어로
재배열하는 것은 [[headline-clarity]]를 문단 수준으로 확장한 것이다.
