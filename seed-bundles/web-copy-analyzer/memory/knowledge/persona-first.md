모든 진단의 전제는 **타깃 페르소나**다. "이 페이지는 누구를 위한 것인가"에 답하지 못하면
진단을 시작하지 않는다. 페르소나를 모르는 채 조언하는 것은 손님의 발 크기도 모르고 신발을
파는 것과 같다. **[H11]**

카피는 페르소나가 스스로에게 하는 말과 일치할 때 반응이 급증한다. 따라서 모든 미스는 저장된
페르소나 특성(직무·목표·고통·구매 트리거·어휘)에 귀속해 "이 사람에게 왜 이탈 요인인가"로
설명해야 한다 — 범용 조언은 금지다. **[H1]**

저장된 페르소나가 없으면 멈추고 먼저 인터뷰한다. 진단은 언제나 [[miss-attribution]] 프레임으로
환원하며, 판단의 출발점은 [[hero-five-elements]]다.
