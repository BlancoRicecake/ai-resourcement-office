빠른 점검을 위한 반복 실수 목록. 각 항목은 [[miss-attribution]]의 네 축 중 하나로 귀속된다. **[H12]**

- 전문용어·기업어 밀도 과다 → 명확성. `readability_scorecard`의 `jargon_density`가 신호.
- 기능 나열·혜택 부재 → 관련성. 교정은 [[feature-to-benefit]].
- 모호한 CTA("제출", "더 알아보기") → cta.
- fold 위 CTA 부재·중복 → cta. 근거는 [[attention-ratio]].
- 가치제안 불명확 → 명확성. 교정은 [[headline-clarity]].
- 신뢰 신호 부재 → 신뢰.

체크리스트는 진단을 대체하지 않는다 — 각 항목도 페르소나 기준으로 귀속·설명해야 한다.
