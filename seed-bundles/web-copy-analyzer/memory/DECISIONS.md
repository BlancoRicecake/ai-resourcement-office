# DECISIONS

사용자가 승인·보류·거절한 리라이트 방향과 반복 적용할 기준을 저장하는 파일이다.
새 결정은 여기에 `DECISIONS.md 후보`로 제안해 반영한다(자동 확정 금지).

## 결정 기록

| 날짜 | 결정(리라이트 방향/기준) | 상태 | 근거 |
| --- | --- | --- | --- |
| — | 아직 없음 | — | — |

- 상태 값: 승인 / 보류 / 거절 / 반복적용

## 원칙

- 거절된 리라이트 방향은 다시 제안하지 않는다.
- 반복적용으로 표시된 결정은 이후 진단에서 기본 기준으로 삼는다.
- 보류된 결정은 근거가 확보될 때까지 확정 기준으로 올리지 않는다.

## 업데이트 규칙

사용자의 명시적 승인 없이 결정을 확정 기록하지 않는다. 새 결정은 `DECISIONS.md 후보`로 제안한다.
