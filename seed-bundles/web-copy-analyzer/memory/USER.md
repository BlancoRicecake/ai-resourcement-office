# USER

사용자의 선호·브랜드 보이스·금지 표현·톤을 저장하는 파일이다. 사용자가 선호·교정을
표현하거나("우리 브랜드는 존댓말", "그 표현 쓰지 마") 리라이트 제안이 채택·거부될 때마다
여기에 `USER.md 후보`로 제안해 반영한다(자동 확정 금지).

## 브랜드 보이스

- 기본값: 페르소나 어휘·톤에 맞춘 you-언어 우선(you:we ≈ 2:1). 명료성을 영리함보다 우선.
- 사용자 지정 보이스: 아직 없음. (예: "존댓말 사용", "친근한 반말", "전문가 톤")

## 톤·산출물 취향

- 아직 기록된 취향 없음. (예: 리포트 길이, before/after 표 형식 선호, 리라이트 대안 개수)

## 금지 표현

- 아직 기록된 금지 표현 없음. (사용자가 "그 표현 쓰지 마"라고 하면 여기에 후보로 제안)

## 업데이트 규칙

새로운 선호·보이스·금지 표현이 관찰되면 바로 확정하지 말고 `USER.md 후보`로 제안한다.
무엇을 후보로 제안하는지 항상 사용자에게 표시한다.
