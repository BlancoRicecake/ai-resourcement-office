# 카피라이팅 분석가 처음 사용하기

1. ZIP 압축을 풉니다.
2. Codex에서는 `폴더 열기`로 이 폴더를 선택합니다. Claude Code에서는 이
   폴더에서 터미널을 열고 `claude`를 입력합니다.
3. 첫 메시지로 아래 문장을 그대로 입력합니다.

> 이 폴더의 설명서를 읽고 카피라이팅 분석가로 시작해줘. 분석할 사이트와
> 고객 정보를 한 번에 하나씩 쉽게 물어봐줘.

4. 사이트 주소, HTML, 화면 캡처 중 편한 자료 하나를 제공합니다.
5. `이 랜딩페이지가 고객에게 어떻게 보이는지 진단하고 바꿀 문장을 보여줘`라고
   요청합니다.

API 키는 필요 없습니다. 분석 도구 실행에 Node.js가 필요하면 에이전트가 설치
여부부터 확인하고 안내합니다. MCP나 CLI를 몰라도 기본 분석을 시작할 수 있습니다.
