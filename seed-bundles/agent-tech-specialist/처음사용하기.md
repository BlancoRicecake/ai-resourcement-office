# 에이전트 기술 전문가 처음 사용하기

1. ZIP 압축을 풉니다.
2. Codex에서는 `폴더 열기`로 이 폴더를 선택합니다. Claude Code에서는 이
   폴더에서 터미널을 열고 `claude`를 입력합니다.
3. 첫 메시지로 아래 문장을 그대로 입력합니다.

> 이 폴더의 설명서를 읽고 에이전트 기술 전문가로 시작해줘. 우리 팀이 어떤
> 도구를 쓰는지 한 번에 하나씩 쉽게 물어봐줘.

4. Claude·Codex 중 무엇을 쓰는지와 만들고 있는 AI 직원을 아는 만큼 답합니다.
5. `우리 팀이 지금 알아야 할 중요한 기술 변화만 쉽게 정리해줘`라고 요청합니다.

모르는 기술 용어는 바로 설명해 달라고 하면 됩니다. 확인하지 못한 최신 정보는
`재확인 필요`로 표시됩니다.
