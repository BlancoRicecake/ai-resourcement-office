# skills

에이전트 기술 전문가가 업무에서 따르는 규칙 모음이다.

- `tech-radar-rules.md`: 기술 추적 규칙 (추적 범주, 신호 강도 판정, 성숙도 단계, 팀 적용 가능성 평가, 소음 걸러내기)
- `advisory-rules.md`: 기술 자문 규칙 (도구 요구 도출 → 후보 비교 → 추천안, 에이전트 메이커 도구 인터뷰 연결)
- `refresher-training-rules.md`: 보수 교육 제안 규칙 (발동 조건, 제안서 형식, 자동 수정 금지)
- `source-credibility-rules.md`: 기술 동향 소스의 공신력 판단 규칙 (공식 문서 > 개발사 블로그 > 커뮤니티, 루머 걸러내기)

이 규칙들은 `worker/agent.md`의 3대 업무(동향 브리핑 / 기술 자문 / 보수 교육 제안)를 구체화한다.
