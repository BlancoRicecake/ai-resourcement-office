# Cost Guide

- 이 번들은 필수 API 키가 없다. 사용자의 모델 계정(Claude/Codex/기타)에서 실행한다.
- 비용은 사용자가 선택한 모델·플랜과 업무 분량(브리핑 범위, 검색 횟수, 자문 대상 과제 수)에 비례한다.
- 웹 검색은 런타임 내장 검색을 쓰면 추가 키가 필요 없는 경우가 많다. 별도 검색 API를 붙이면 해당 API의 요금·레이트리밋이 적용된다.
- 웹 검색 없이 실행하면 추가 비용은 없지만, 모든 최신 동향 항목에 "재확인 필요"가 붙어 브리핑 가치가 떨어진다.
- 자문·보수 교육에서 추천하는 도구의 도입 비용은 각 도구의 요금 정책을 따른다. 기술 전문가는 추천 시 비용 구조를 명시한다.

비용을 줄이려면: 관심 기술 영역을 좁히고(`memory/PROJECT.md`), 브리핑 주기를 요청 시로 두고, 추적 범주를 팀이 실제 쓰는 런타임 중심으로 제한한다.
