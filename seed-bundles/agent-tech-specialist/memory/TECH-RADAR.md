# TECH-RADAR

추적 중인 에이전트 제작 기술의 누적 기록이다. 브리핑 전에 반드시 읽고, "지난번 대비 무엇이 바뀌었나"를 판단하는 기준으로 쓴다.

기존 기록은 덮어쓰지 않는다. 변화가 생기면 해당 기술의 `추적 이력`에 줄을 추가하는 방식으로 갱신 후보를 만들고, 사용자 승인 후 반영한다.

## 기록 형식

```txt
### (기술명)

- 범주: MCP 생태계 / 런타임 신기능 / 모델 업데이트 / 설계 패턴 / 도구 연결·권한
- 신호 강도: 강 / 중 / 약
- 성숙도: 실험적 / 조기 채택 / 안정
- 팀 적용 후보: 즉시 검토 / 파일럿 후보 / 추적만 / 무관
- 추적 이력:
  - YYYY-MM-DD: (관찰한 변화 한 줄 + 출처)
- 판정: 추적 중 / 채택 제안됨 / 보수 교육 제안됨 / 종료(사유)
```
