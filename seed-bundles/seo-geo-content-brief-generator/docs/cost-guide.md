# Cost Guide

## 모의(mock) 모드

`OPENAI_API_KEY`가 없으면 템플릿 기반으로 동작하며 **비용이 전혀 들지
않습니다.** 먼저 이 모드로 브리프 구조를 확인하는 것을 권장합니다.

## LLM 모드 (기본 모델: gpt-4o-mini)

- 브리프 1건: 약 $0.01-$0.08
- 입력(키워드/제품 설명)은 짧고 출력(브리프 본문)이 길기 때문에 출력 토큰이
  비용의 대부분을 차지합니다.
- 긴 제품 설명이나 다수 키워드 처리 시 비용이 증가할 수 있습니다.
- 정확한 단가는 OpenAI 공식 가격 페이지에서 확인하세요.
- 생성 완료 후 UI에 실제 사용 토큰 수가 표시됩니다.

## 권장 사항

- 키워드는 1-3개부터 테스트한다.
- 대량 브리프 생성 전 출력 품질을 먼저 확인한다.
- 모델 제공자 대시보드에서 사용량 한도(hard limit)를 설정한다.
