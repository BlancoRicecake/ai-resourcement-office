# Cost Guide

## 무료 템플릿 모드

`OPENAI_API_KEY`가 없으면 로컬 템플릿 기반으로 동작하며 비용이 들지 않습니다.
먼저 이 모드로 구조와 입력 흐름을 확인하는 것을 권장합니다.

## LLM 모드

- 기본 모델은 `gpt-4o-mini`입니다.
- 긴 스크립트일수록 출력 토큰이 늘어 비용도 증가합니다.
- 정확한 단가는 모델 제공자의 공식 가격 페이지에서 확인하세요.
- 생성 완료 후 UI에 실제 사용 토큰 수가 표시됩니다.

## 권장 사항

- 러닝타임과 출력 모드를 먼저 좁힌다.
- 브랜드 문구와 CTA는 사람이 최종 확인한다.
- 모델 제공자 대시보드에서 사용량 한도를 설정한다.
