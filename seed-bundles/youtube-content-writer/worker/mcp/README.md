# MCP Notes

이 번들은 기본 실행에 MCP가 필요하지 않습니다.

나중에 연결할 수 있는 MCP 예시:

- Google Drive 또는 Dropbox: 영상 기획 문서 읽기
- Notion: 콘텐츠 캘린더와 브랜드 가이드 읽기
- YouTube Data API: 기존 영상 제목과 성과 메모 참고

연결 전 확인할 것:

- 읽기 권한만 필요한지, 쓰기 권한도 필요한지 구분한다.
- API 키와 OAuth 토큰은 `.env` 또는 사용자의 시크릿 저장소에 둔다.
- 원본 영상 대본이나 경쟁 채널 대본을 그대로 복제하지 않는다.
