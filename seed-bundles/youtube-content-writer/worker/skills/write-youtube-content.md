# Skill: 유튜브 컨텐츠 원고 작성

유튜브 컨텐츠 작가가 영상 아이디어를 받아 촬영 가능한 영상 원고를 만들 때
따르는 절차.

## 절차

1. 영상 주제, 타깃 시청자, 영상 형식, 말투, 러닝타임, CTA 목적을 확인한다.
2. 정보가 부족하면 다음 질문을 먼저 한다:
   - 어떤 시청자를 설득하거나 도와야 하나요?
   - 영상 형식은 Tutorial, Listicle, Opinion, Case Study, Product Demo 중 무엇인가요?
   - 완성 원고와 토킹포인트 중 어떤 결과가 필요한가요?
   - 마지막 CTA는 구독, 상담, 구매, 뉴스레터, 다운로드 중 무엇인가요?
3. 미니 SaaS CLI로 초안을 생성한다:

   ```bash
   python app/app.py --cli <입력.md> -o <영상원고.md>
   ```

4. 생성된 초안을 읽고 다음을 점검한다:
   - 첫 15초 안에 볼 이유가 분명한가
   - 형식별 구조가 실제로 다르게 적용됐는가
   - 섹션별 예상 시간이 전체 러닝타임과 맞는가
   - B-roll, 화면 캡처, 자막 포인트가 들어 있는가
   - CTA가 사용자의 비즈니스 목적과 연결되는가
5. 검증되지 않은 수치나 사례가 있으면 `[검증 필요]` 표시를 유지한다.

## 주의

- 특정 유튜버의 대본을 그대로 모방하지 않는다.
- 사용자가 제공한 브랜드 톤은 반영하되 허위 후기나 과장 광고는 만들지 않는다.
- API 키가 없으면 무료 템플릿 모드로 실행된다. 결과 전달 시 모드를 명시한다.
