# SCRIPTS

과거 원고 이력과 반응이 좋았던 훅, 구조, 표현 패턴, 사용자 피드백에서 배운 점을 기록하는 파일이다.

## 기록 원칙

- 원고를 완성할 때마다 어떤 형식, 어떤 훅, 어떤 CTA를 썼는지 남긴다.
- 사용자 피드백이나 시청 반응 데이터가 있으면 함께 기록한다.
- 반응 근거가 없는 패턴은 `검증 필요`로 표시하고 확정 패턴으로 올리지 않는다.

## 기록 형식

```md
## YYYY-MM-DD 영상 제목 또는 주제

- 영상 형식:
- 타깃 시청자:
- 출력 형태(완성 원고 / 토킹포인트):
- 사용한 훅:
- 구조 요약:
- CTA:
- 사용자 피드백:
- 시청 반응(있으면):
- 배운 점:
- 다음 원고 반영:
```

## 유효한 훅 패턴

- 아직 없음.

## 유효한 구조/표현 패턴

- 아직 없음.

## 원고 기록

- 아직 기록된 원고 없음.

## 업데이트 규칙

기록은 자동 확정하지 않고 `SCRIPTS.md 후보`로 제안한 뒤 사용자 승인 후 반영한다.
