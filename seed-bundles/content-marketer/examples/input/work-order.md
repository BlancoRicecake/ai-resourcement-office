# 예시 작업 지시서 (연동 모드)

그로스 마케터가 넘긴 성장 실험 설계서에서 추출한 부분이다.

## 실험 맥락

- 실험명: 4주 무료 체험 신청 증가 실험
- 가설: 초기 스타트업 PM/창업자가 겪는 `고객 인터뷰 정리 부담`을 직접 건드리는 문제 해결형 콘텐츠를 만들고, 글 끝과 링크드인 배포문에서 무료 체험 CTA를 명확히 제시하면 무료 체험 신청이 증가한다.
- 타깃: 초기 스타트업의 PM, 창업자
- 핵심 메시지: 고객 인터뷰는 많이 하는데 팀이 다시 읽고 실행으로 옮기지 못한다면, 인터뷰 기록을 요약 가능한 지식 자산으로 바꿔야 한다.
- 채널: 블로그, 링크드인
- 1차 KPI: 무료 체험 신청 수
- 실험 기간: 4주
- 브랜드 톤: 전문가형이되 초기 창업자에게 친근함. 금지 표현: 과장된 효능 단정.
