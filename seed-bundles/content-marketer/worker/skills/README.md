# Skills

콘텐츠 마케터가 업무 단계별로 따르는 규칙 문서다.

- `editorial-planning.md`: 콘텐츠 결(필러·에디토리얼 캘린더·주제 파이프라인) 설계. 개별 콘텐츠 배치 전에 참고한다.
- `research-and-evidence.md`: 검색 의도·고객 언어·출처·주장 검증표를 정리하는 리서치 규칙. 초안 전에 적용한다.
- `channel-format-rules.md`: 블로그/SEO·인스타그램·스레드·링크드인·뉴스레터 채널별 작성 규격. 초안 작성 단계에서 따른다.
- `tone-options.md`: 브랜드 보이스 기록이 없을 때 제안하는 톤 선택지. 톤 확정 단계에서 쓴다.
- `revision-rules.md`: 피드백을 위치 특정해 부분 수정하는 규칙. 피드백 반영 단계에서 따른다.
- `claim-safety-rules.md`: 허위 콘텐츠 금지, 컨펌 게이트, `[검증 필요]` 표기 규칙. 항상 지킨다.
- `publish-connectors.md`: 발행 tier와 채널별 커넥터 세팅 안내. 사용자가 자동 발행을 원할 때 따른다.
- `growth-marketer-interface.md`: 그로스 마케터 작업 지시서 수신/결과 반환 계약. 연동 모드에서 따른다.
