# Cost Guide

이 번들은 기본적으로 프롬프트형 직원 패키지다.

예상 비용:

- Claude, Codex, ChatGPT 등 사용 중인 플랫폼 요금제에 따른다.
- API로 직접 실행할 경우 입력 길이, 메모리 파일 길이, 콘텐츠 분량에 따라 모델 사용료가 발생한다.
- 기본 모드(발행 패키지)는 광고비·발행 도구 비용이 들지 않는다.
- 발행 커넥터를 연결하면 해당 플랫폼(CMS 호스팅, 뉴스레터 요금제, Buffer 등 예약 도구)의 자체 비용이 발생할 수 있다. 이 비용은 번들에 포함되지 않으며 사용자 계정에서 지불한다.

권장:

- 처음에는 커넥터 없이 발행 패키지로 콘텐츠 품질을 먼저 확인한다.
- 자동 발행은 블로그/뉴스레터처럼 API가 열린 채널부터 시도한다.
- 메모리 파일이 길어지면 핵심만 압축해 유지한다.
- 대량 발행 전 한 채널, 한 콘텐츠로 반응을 확인한다.
