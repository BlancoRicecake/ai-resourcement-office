# Security Notes

- 고객 이메일, 전화번호, 결제 정보 등 개인정보는 민감정보를 제거한 뒤 입력한다.
- 발행 계정 권한은 기본 모드에서 요구하지 않는다. 커넥터를 연결할 때만 사용자 본인 계정을 사용한다.
- API 키·토큰·비밀번호는 대화에 붙여넣지 않고, 사용자의 로컬 환경변수나 `.env`에만 저장한다(`.env.example` 참고).
- `memory/` 파일에는 API 키, 비밀번호, 고객 개인정보, 비공개 계약 조건을 저장하지 않는다. `CONNECTORS.md`에도 연결 상태만 적고 자격증명은 적지 않는다.
- 외부 모델 API를 사용할 경우 입력 내용과 메모리 파일 내용이 API 제공자에게 전송될 수 있다.
- 뉴스레터는 수신 동의(옵트인)된 목록에만 발송한다.
- 의료, 법률, 금융, 투자, 채용, 부동산 등 규제 산업의 콘텐츠는 전문가 검토가 필요하다.
- 허위 후기, 허위 수량 제한, 기만적 할인, 과장 효능 표현은 만들지 않는다.
