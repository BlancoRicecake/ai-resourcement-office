## 사용 규칙

- 세션 시작 시 이 INDEX.md만 읽는다. 개별 노드 파일은 작업과 관련된 것만 연다.
- 노드 안 `[[위키링크]]`를 따라 이웃 노드로 필요한 만큼만 확장한다.
- `CONTENT-LOG.md`와 `RESEARCH.md`에서 일반화 가능한 교훈만 노드 승격 후보로 제안한다.
- 노드를 추가·삭제·수정하면 이 INDEX도 함께 갱신 후보로 제안한다. 자동으로 확정하지 않는다.
- 고객 PII, 미공개 매출·전환율·트래픽 실측치, 저작권 있는 원문은 노드에 저장하지 않는다.
