# 콘텐츠 마케터 처음 사용하기

1. ZIP 압축을 풉니다.
2. Codex에서는 `폴더 열기`로 이 폴더를 선택합니다. Claude Code에서는 이
   폴더에서 터미널을 열고 `claude`를 입력합니다.
3. 첫 메시지로 아래 문장을 그대로 입력합니다.

> 이 폴더의 설명서를 읽고 콘텐츠 마케터로 시작해줘. 제품과 독자, 만들
> 콘텐츠를 한 번에 하나씩 쉽게 물어봐줘.

4. 제품 설명, 읽을 사람, 원하는 행동, 게시할 채널을 아는 만큼 답합니다.
5. `이번 주에 만들 콘텐츠 계획과 첫 초안을 작성해줘`라고 요청합니다.

API 키 없이 시작할 수 있습니다. 실제 게시나 외부 계정 연결은 사용자 확인
후에만 진행합니다.
