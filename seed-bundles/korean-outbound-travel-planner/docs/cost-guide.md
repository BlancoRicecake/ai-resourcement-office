# Cost Guide

- FastAPI와 SQLite 로컬 실행 자체에는 별도 사용료가 없다.
- OpenAI, Amadeus 프로덕션, Google Places·Routes는 사용량과 계정 조건에 따라 비용이 발생할 수 있다.
- Google Routes Matrix는 출발지 수 × 목적지 수의 요소 단위로 과금될 수 있으므로 앱은 일정 후보를 12개로 제한한다.
- `GET /api/providers/health`에서 실제 연결 여부를 확인한다.
- 공급자 콘솔에서 일일 할당량과 예산 알림을 설정한다.
- API 키가 없으면 외부 비용 없이 샘플 모드를 시험할 수 있으나 실제 예약 판단에 사용하면 안 된다.

## 여행비 합산 범위

- 일정 총액은 선택 항공 총액 + 선택 숙소 총액 + 관광·식사·카페 + 일정 내 현지 이동비의 합이다.
- 장소비는 공급자의 1인 추정액에 성인 수와 아동 70% 환산 인원을 곱한다.
- 현지 이동비는 도보 0원, 대중교통 탑승 1회당 일본 2,000원·미국 3,500원 상당의 설정값을 사용한다.
- 렌터카 선택 시 일반 차량 1일 90,000원, 스포츠 차량 1일 140,000원의 계획값을 사용한다. 실제 보험·세금·보증금·차종 보장 가격은 공식 예약 화면에서 확인한다.
- 유류비는 숙소↔첫·마지막 장소와 장소 간 거리, 일반 차량 11km/L·스포츠 차량 9.3km/L, 미국 2,100원/L·일본 1,750원/L 계획값으로 계산한다.
- 장소와 숙소에 확인된 계획용 주차비를 현지 이동 총액에 포함하고, 미상 주차비는 별도 확인 항목으로 남긴다.
- 공항↔숙소, 보험, eSIM, 쇼핑과 공급자가 포함 여부를 확정하지 않은 세금·리조트피는 합계에서 제외하고 화면에 별도로 고지한다.
- 지도 선은 장소 좌표를 방문 순서대로 연결한 개요이며 실제 도로 운임이나 환승 경로를 의미하지 않는다.
