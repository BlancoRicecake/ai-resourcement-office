# Security Notes

- `.env`를 커밋하지 말고 `.env.example`만 공유한다.
- 모든 공급자 키는 FastAPI 서버에서만 읽으며 HTML과 JavaScript에 삽입하지 않는다.
- Google 키는 필요한 API와 서버 환경으로 제한한다.
- 여권번호, 결제정보, 항공사·숙박 계정 비밀번호를 수집하지 않는다.
- 취향 저장은 명시적 동의가 있을 때만 수행하며 `DELETE /api/profile`로 삭제한다.
- 로컬 SQLite 파일에는 여행 조건이 저장될 수 있으므로 공유 PC에서는 프로필 저장을 사용하지 않는다.
- 로그에 요청 헤더, 토큰, 원본 공급자 오류 본문을 출력하지 않는다.
- 외부 예약 링크의 도메인과 최종 가격을 사용자가 직접 확인해야 한다.
- 공개 배포는 `scripts/build_release.py`의 허용 목록을 사용한다. 작업 폴더 전체를 압축하지 말고 `.env`, `.venv`, `app/data`, DB, 로그와 캐시가 제외되었는지 확인한다.
- 신규 목적지 조사 팩은 출처 URL과 상태만 저장하며 웹 페이지 본문이나 예약 계정 정보를 저장하지 않는다.
