# Limitations

- 품질 검증 도시는 도쿄·오사카·뉴욕·로스앤젤레스다. 다른 목적지는 조사 팩과 검증 링크를 제공하지만 최종 동선 계산은 검증 전까지 제한한다.
- API가 반환하는 전체 시장을 검색하지 않으므로 보편적인 최저가를 보장하지 않는다.
- Amadeus 테스트 환경의 응답은 실제 프로덕션 재고가 아닐 수 있다.
- Booking.com과 Skyscanner는 파트너 승인과 별도 계약 없이는 활성화되지 않는다.
- Airbnb는 실시간 API를 사용하지 않으며 검색 링크의 가격·재고는 미검증이다.
- Google Places의 영업시간·접근성·리뷰 필드는 누락되거나 변경될 수 있다.
- 경로는 실시간 장애, 파업, 공사, 행사와 개인 보행 속도를 완전히 반영하지 못한다.
- 자동 가격 감시, 지연 알림, 예약·결제·취소, 계정 동기화는 포함하지 않는다.
- 로컬 앱은 일반 검색엔진 결과를 자동 수집하거나 스크래핑하지 않는다. 호스트 에이전트에 웹 검색 도구가 없으면 공식·비교 플랫폼 링크만 제공한다.
- `guided_research` 모드의 항공·숙소 금액은 일정 예산 계산용 추정치이며 가격·재고·수하물·취소 조건을 보장하지 않는다.
- 비자·입국·안전 정보는 참고 링크이며 최종 판단은 공식 기관과 사용자에게 있다.
- 독립 실행 앱에는 범용 웹 브라우저가 내장되어 있지 않다. 호스트 에이전트의 웹 검색 도구가 없으면 신규 목적지 정보는 자동 수집되지 않고 조사 체크리스트와 링크로 남는다.
