# 도쿄 여행 설계 결과 예시

> 이 파일은 출력 구조 예시입니다. 표시 가격과 재고는 실제 검색 결과가 아닙니다.

## 현재 옵션

- 추천 항공: 총액, 수하물, 환승, 총 소요시간과 조회 시각 표시
- 추천 숙소: 1객실 총 숙박비, 위치, 평점, 취소 조건과 재고 검증 상태 표시
- 확인 필요: 예약 직전 총액·수하물·취소 조건, 일본 공식 입국 정보, 당일 영업시간

## 균형형

- 장점: 대표 명소·식사·카페와 이동 효율의 균형
- 단점: 인기 장소의 혼잡 가능성
- 1일차: 공항 지연 버퍼를 둔 오후 일정
- 2일차: 명소 → 점심 → 카페 → 저녁 동선
- 3일차: 근거리 명소와 공항 이동 버퍼

## 비용 절약형

- 장점: 무료 명소와 저가 옵션 중심
- 단점: 환승 또는 이동시간 증가 가능

## 여유형

- 장점: 장소 수와 이동 부담 감소
- 단점: 방문 장소가 적고 편의성 우선 옵션의 비용이 높을 수 있음
