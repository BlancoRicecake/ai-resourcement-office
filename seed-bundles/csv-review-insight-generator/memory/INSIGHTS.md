# INSIGHTS

과거 리뷰 분석에서 발견한 패턴, 반복 인사이트, 배운 점을 기록하는 파일이다.

## 기록 원칙

- 한 번의 분석 결과가 아니라 여러 분석에서 반복 확인된 패턴을 우선 기록한다.
- 실제 리뷰 내용으로 확인된 것만 기록하고, 추정은 `가설`로 표시한다.
- 실행 모드(mock 모드/LLM 모드)를 함께 적어 결과의 신뢰 수준을 구분한다.

## 기록 형식

```md
## YYYY-MM-DD 분석명

- 분석 대상:
- 데이터 규모:
- 실행 모드:
- 감성 분포:
- 반복된 불만:
- 반복된 칭찬:
- 개선 요청:
- 이전 분석과 비교:
- 배운 점:
- 다음 분석 반영:
```

## 인사이트 기록

- 아직 기록된 인사이트 없음.
