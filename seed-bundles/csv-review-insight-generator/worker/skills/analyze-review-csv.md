# Skill: 리뷰 CSV 분석 실행

리뷰 분석 직원이 CSV 파일을 받아 리포트를 생성할 때 따르는 절차.

## 절차

1. 입력 CSV의 행 수와 컬럼 구조를 먼저 확인한다.
2. 200행을 초과하면 사용자에게 예상 비용을 알리고 계속할지 확인한다.
3. 미니 SaaS CLI로 분석을 실행한다:

   ```bash
   python app/app.py --cli <입력.csv> -o <리포트.md>
   ```

4. 생성된 리포트를 읽고 다음을 점검한다:
   - 감성 분포 합계가 전체 리뷰 수와 일치하는가
   - 주요 불만/칭찬 항목이 실제 리뷰 내용과 부합하는가
5. 점검 결과와 함께 리포트를 사용자에게 전달한다.

## 주의

- API 키가 없으면 자동으로 모의(mock) 모드로 실행된다. 결과 전달 시 모드를 명시한다.
- 개인정보(이름, 전화번호, 이메일, 주문번호)가 보이면 실행을 멈추고
  사용자에게 익명화를 먼저 요청한다.
