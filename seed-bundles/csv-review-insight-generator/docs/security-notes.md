# Security Notes

## 데이터 흐름

- **모의(mock) 모드**: 모든 처리가 로컬에서 수행되며 데이터가 PC 밖으로
  나가지 않습니다.
- **LLM 모드**: 리뷰 텍스트 전체가 OpenAI API로 전송됩니다. 제3자 API 약관과
  데이터 처리 정책을 먼저 확인하세요.

## 개인정보

- 리뷰에 이름, 전화번호, 이메일, 주문번호가 포함될 수 있습니다.
- 실제 운영 데이터는 익명화 후 분석하는 것을 권장합니다.
- 워커 스킬에도 개인정보 발견 시 실행을 멈추고 익명화를 요청하는 규칙이
  포함되어 있습니다 (`worker/skills/analyze-review-csv.md`).

## API 키 관리

- 키는 번들 루트의 `.env` 파일에만 저장하세요 (`.env.example` 참고).
- `.env`는 절대 커밋하거나 공유하지 마세요.
- 키가 노출되면 즉시 OpenAI 대시보드에서 폐기하고 재발급하세요.

## 네트워크 노출

- 웹 UI 서버는 `127.0.0.1`(localhost)에만 바인딩되어 같은 PC에서만 접근
  가능합니다.
- 외부 공개(0.0.0.0 바인딩, 포트포워딩, 터널링)는 지원 범위가 아니며 인증이
  없으므로 권장하지 않습니다.
