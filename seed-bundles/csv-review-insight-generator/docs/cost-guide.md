# Cost Guide

## 모의(mock) 모드

`OPENAI_API_KEY`가 없으면 키워드 규칙 기반으로 동작하며 **비용이 전혀 들지
않습니다.** 먼저 이 모드로 앱 동작을 확인하는 것을 권장합니다.

## LLM 모드 (기본 모델: gpt-4o-mini)

리뷰 전체가 한 번의 API 요청으로 전송됩니다. 대략적인 추정:

| 규모 | 입력 토큰(추정) | 예상 비용 |
| --- | --- | --- |
| 샘플 20행 (한 줄 리뷰) | ~1,500 | $0.01 미만 |
| 200행 (행 제한 기본값) | ~15,000 | $0.01-$0.05 |
| 1,000행 이상 | 행 제한 상향 필요 | 모델/프롬프트 길이에 따라 증가 |

- 한국어는 영어보다 토큰 소모가 큽니다 (대략 글자 2자당 1토큰으로 추정).
- 정확한 단가는 OpenAI 공식 가격 페이지에서 확인하세요. 모델 단가는 수시로
  바뀝니다.
- 분석 완료 후 UI에 실제 사용 토큰 수가 표시됩니다.

## 비용 보호 장치

- `MAX_REVIEW_ROWS`(기본 200행)를 초과하는 행은 잘려나가며 UI에 표시됩니다.
- 상향하려면 환경변수로 조정: `MAX_REVIEW_ROWS=1000 python app.py`

## 권장 사항

- 샘플 CSV로 먼저 테스트한다.
- 모델 제공자 대시보드에서 사용량 한도(hard limit)를 설정한다.
- 정기 실행(스케줄러)에 편입하기 전에 1회 실행 비용을 확인한다.
