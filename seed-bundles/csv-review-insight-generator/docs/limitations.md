# Limitations

- 짧거나 반어적인 리뷰는 분류가 부정확할 수 있다 (특히 모의 모드).
- 모의(mock) 모드는 사전 정의된 한국어 키워드 규칙에 의존하므로 영어 리뷰나
  도메인 특수 용어에는 약하다.
- LLM 모드는 리뷰 전체를 한 번의 요청으로 보내므로 매우 큰 CSV(수천 행)는
  모델 컨텍스트 한도에 걸릴 수 있다. 기본 행 제한은 200행이다.
- 대용량 CSV는 모델 비용이 증가한다.
- 감성 분류 결과는 참고 자료이며, 생성된 인사이트는 사용자가 검토해야 한다.
- 개인정보가 포함된 리뷰는 처리 전에 제거해야 한다.
