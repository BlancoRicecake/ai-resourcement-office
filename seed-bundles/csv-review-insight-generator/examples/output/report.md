# 리뷰 분석 리포트

총 20건 리뷰 분석 (모의 모드: 키워드 기반 규칙 분석)

## 감성 분포

- 긍정: 10건 (50%)
- 부정: 9건 (45%)
- 중립: 1건 (5%)

## 부정 리뷰 주요 주제

- 배송: 4건
- 포장: 2건
- 환불/교환: 2건
- 품질: 1건
- 고객센터/응대: 1건

## 긍정 리뷰 주요 주제

- 품질: 3건
- 고객센터/응대: 2건
- 가격: 2건
- 배송: 1건
- 포장: 1건

## 참고

- 이 리포트는 OPENAI_API_KEY 없이 실행된 모의(mock) 모드 결과입니다.
- 키워드 규칙 기반이므로 짧거나 반어적인 리뷰는 오분류될 수 있습니다.
- API 키를 설정하면 LLM 기반의 심층 분석 리포트가 생성됩니다.
