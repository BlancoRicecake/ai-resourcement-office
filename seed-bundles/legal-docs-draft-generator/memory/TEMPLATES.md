# TEMPLATES

과거 작성한 문서 유형, 사용자가 승인한 조항 패턴, 반복 사용하는 문구와 구조, 배운 점을 기록하는 파일이다.

## 기록 원칙

- 사용자가 승인한 조항 패턴만 재사용 대상으로 기록한다.
- 조항 패턴은 서비스 실정(무료/유료, 개인정보 수집 여부)과 함께 기록해, 맞지 않는 서비스에 그대로 적용하지 않는다.
- 법령 개정으로 무효화될 수 있으므로, 기록된 패턴도 재사용 전에 최신 법령 확인을 권고한다.
- [대괄호] 미확정 값은 템플릿에도 미확정 상태 그대로 기록한다.

## 기록 형식

```md
## YYYY-MM-DD 문서명

- 문서 유형: (이용약관 / 개인정보처리방침 / 면책조항)
- 서비스 실정: (무료/유료, 개인정보 수집 여부, 대상 지역)
- 생성 모드: (mock / LLM)
- 승인된 조항 패턴:
- 반복 사용할 문구·구조:
- 사용자 피드백:
- 배운 점:
- 재사용 조건:
```
