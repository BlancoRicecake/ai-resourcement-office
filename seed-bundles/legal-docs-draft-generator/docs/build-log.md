# Build Log

## Source Problem

온라인 서비스(SaaS 등)를 시작할 때 이용약관, 개인정보처리방침, 면책조항 등
필수 법률 문서가 필요하지만, 초기 창업자·1인 개발자는 어디서부터 시작할지
모르고 변호사 선임 전 초안조차 없는 경우가 많다.

## MVP Scope

서비스 정보를 받아 상황(무료/유료, 개인정보 수집 여부)에 맞는 법률 문서
초안을 생성한다. 법률 자문이 아닌 "변호사 검토용 초안"으로 포지셔닝한다.

## 2026-07-06: v0.1 구현 완료

- `app/app.py`: Python 표준 라이브러리만 사용하는 단일 파일 로컬 웹앱 (포트 8790).
  - 웹 UI + CLI 모드, 갱신된 bundle-template 패턴 적용 (템플릿 첫 실전 사용).
  - 상황 분기: 유료 여부에 따라 결제·청약철회 조항(전자상거래법) 생성/생략,
    개인정보 수집 여부에 따라 방침 구조 전환 (수집 시 개인정보 보호법 제30조
    필수 기재사항 구조, 미수집 시 미수집 방침 + 인프라 자동 수집 고지).
  - "법률 자문 아님" 고지를 UI, 산출물, LLM 프롬프트 3중으로 내장.
    LLM 응답에 고지가 빠지면 코드에서 강제 삽입.
- `worker/`: 게시 가능 여부 질문에 단정 금지, 해외 법제(GDPR 등) 별도 검토
  안내 규칙 포함.
- `examples/`: AI력 사무소 자체를 샘플 입력으로 사용 (실제 CLI 실행 결과).
