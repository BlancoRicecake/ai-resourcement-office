# Security Notes

## 데이터 흐름

- **모의(mock) 모드**: 모든 처리가 로컬에서 수행되며 데이터가 PC 밖으로
  나가지 않습니다.
- **LLM 모드**: 입력한 서비스 정보가 OpenAI API로 전송됩니다.

## 입력 주의

- 미공개 서비스의 사업 계획, 내부 전략, 미출시 기능 상세를 입력하지 않도록
  주의하세요. 법률 문서 생성에는 공개 가능한 수준의 서비스 개요면 충분합니다.

## API 키 관리

- 키는 번들 루트의 `.env` 파일에만 저장하세요 (`.env.example` 참고).
- `.env`는 절대 커밋하거나 공유하지 마세요.

## 네트워크 노출

- 웹 UI 서버는 `127.0.0.1`(localhost)에만 바인딩되어 같은 PC에서만 접근
  가능합니다. 외부 공개는 지원 범위가 아닙니다.
