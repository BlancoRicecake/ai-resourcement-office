# Cost Guide

## 모의(mock) 모드

`OPENAI_API_KEY`가 없으면 대한민국 법령 구조 기반 템플릿으로 동작하며
**비용이 전혀 들지 않습니다.** 표준적인 서비스라면 mock 모드 초안만으로도
변호사 검토의 출발점으로 충분할 수 있습니다.

## LLM 모드 (기본 모델: gpt-4o-mini)

- 문서 3종 세트 1회: 약 $0.05-$0.25
- 법률 문서는 출력이 길어 출력 토큰이 비용의 대부분을 차지합니다.
- 정확한 단가는 OpenAI 공식 가격 페이지에서 확인하세요.
- 생성 완료 후 UI에 실제 사용 토큰 수가 표시됩니다.

## 권장 사항

- mock 모드로 구조를 먼저 확인한 뒤 필요하면 LLM 모드로 재생성한다.
- 생성 비용보다 훨씬 큰 비용은 변호사 검토 비용이며, 이는 생략하면 안 되는
  비용이다.
