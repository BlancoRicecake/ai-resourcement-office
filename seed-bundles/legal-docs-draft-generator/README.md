# 온라인 서비스 법률 문서 초안 생성기

서비스 정보(이름, 설명, 유료 여부, 개인정보 수집 여부)를 입력하면 이용약관,
개인정보처리방침, 면책조항 초안을 상황에 맞게 생성하는 미니 SaaS 번들입니다.

> ⚠️ **이 패키지는 법률 자문이 아닙니다.** 생성물은 초안이며, 실제 서비스에
> 게시하기 전에 반드시 변호사 등 전문가의 검토를 받아야 합니다.

## Quick Start

Python 3.8+만 있으면 됩니다. pip 설치가 필요 없습니다.

```bash
python app/app.py
```

브라우저에서 `http://127.0.0.1:8790` 이 열립니다.
API 키가 없어도 무료 모의(mock) 모드(대한민국 법령 구조 기반 템플릿)로 바로
체험할 수 있습니다.

LLM 모드를 쓰려면:

1. `.env.example`을 `.env`로 복사
2. `OPENAI_API_KEY=` 뒤에 본인 키 입력
3. 앱 재시작

CLI 실행:

```bash
python app/app.py --cli examples/input/service.md -o legal-docs.md
```

## 상황 맞춤 동작

- **무료 서비스** → 결제/청약철회 조항 미생성, **유료 서비스** → 전자상거래법
  기반 결제·청약철회 조항 포함
- **개인정보 미수집** → 미수집 방침 + 인프라 사업자 자동 수집 고지 구조,
  **수집** → 개인정보 보호법 제30조 필수 기재사항 구조

## AI Worker

법률 문서 초안 직원 — `worker/agent.md` (지시문), `worker/skills/` (작업 절차),
`worker/harness/` (Claude Code 등에서 구동하는 방법)

## 구성

```txt
app/        미니 SaaS (단일 파일 웹앱 + CLI)
worker/     에이전트 지시문, 스킬, 하네스, MCP 가이드
examples/   샘플 입력 + 실제 실행으로 생성한 법률 문서 초안
docs/       빌드 로그, 비용/보안/한계 가이드
```

## User Responsibility

이 패키지는 호스팅 서비스가 아니며 법률 자문도 아닙니다. 사용자는 자신의
환경과 API 키로 직접 실행하고, 생성된 초안은 반드시 전문가 검토를 거쳐
사용해야 합니다. 실행 전 `docs/limitations.md`를 확인하세요.
