# Cost Guide

## 모의(mock) 모드

`OPENAI_API_KEY`가 없으면 템플릿 기반으로 동작하며 **비용이 전혀 들지
않습니다.** 먼저 이 모드로 제안서 구조를 확인하는 것을 권장합니다.

## LLM 모드 (기본 모델: gpt-4o-mini)

- 짧은 제안서 초안 1건: 약 $0.03-$0.20
- 출력(제안서 본문)이 길어 출력 토큰이 비용의 대부분을 차지합니다.
- 긴 참고자료를 넣을수록 입력 비용도 증가합니다.
- 정확한 단가는 OpenAI 공식 가격 페이지에서 확인하세요.
- 생성 완료 후 UI에 실제 사용 토큰 수가 표시됩니다.

## 권장 사항

- 고객 요구사항을 먼저 요약한 뒤 입력한다.
- 계약 문구와 가격은 사람이 검토한다.
- 모델 제공자 대시보드에서 사용량 한도(hard limit)를 설정한다.
