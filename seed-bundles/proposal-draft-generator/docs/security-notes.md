# Security Notes

## 데이터 흐름

- **모의(mock) 모드**: 모든 처리가 로컬에서 수행되며 데이터가 PC 밖으로
  나가지 않습니다.
- **LLM 모드**: 입력한 고객 문제, 범위, 예산 정보가 OpenAI API로 전송됩니다.

## 기밀 정보 주의

- 고객명, 계약 조건, 내부 견적, 미공개 전략 입력에 특히 주의하세요.
- NDA가 있는 고객 정보는 익명화("A사", "제조업 고객" 등) 후 입력하는 것을
  권장합니다.
- 워커 스킬에도 기밀 발견 시 익명화 여부를 먼저 확인하는 규칙이 포함되어
  있습니다 (`worker/skills/draft-proposal.md`).

## API 키 관리

- 키는 번들 루트의 `.env` 파일에만 저장하세요 (`.env.example` 참고).
- `.env`는 절대 커밋하거나 공유하지 마세요.

## 네트워크 노출

- 웹 UI 서버는 `127.0.0.1`(localhost)에만 바인딩되어 같은 PC에서만 접근
  가능합니다. 외부 공개는 지원 범위가 아닙니다.
