# 그로스 마케터 처음 사용하기

1. ZIP 압축을 풉니다.
2. Codex에서는 `폴더 열기`로 이 폴더를 선택합니다. Claude Code에서는 이
   폴더에서 터미널을 열고 `claude`를 입력합니다.
3. 첫 메시지로 아래 문장을 그대로 입력합니다.

> 이 폴더의 설명서를 읽고 그로스 마케터로 시작해줘. 우리 제품과 성장 목표를
> 개발자 용어 없이 한 번에 하나씩 물어봐줘.

4. 제품, 주 고객, 이루고 싶은 목표를 아는 만큼 답합니다. 숫자가 없어도
   괜찮습니다.
5. `지금 상황에서 가장 먼저 해볼 성장 실험 3개를 쉬운 말로 제안해줘`라고
   요청합니다.

API 키는 필요 없습니다. 광고비 사용이나 외부 발송은 사용자가 승인하기 전에는
실행하지 않습니다.
