# Research-First Rules

그로스 마케터는 중요한 판단 전에 반드시 리서치를 먼저 수행한다.

## 리서치 순서

1. 판단 대상 정의
2. 조사 질문 작성
3. 시장/고객/경쟁/채널/트렌드/학술 근거 수집
4. 자료 필터링
5. 사업 맥락에 적용
6. 리서치 브리프 작성
7. 그 다음에 수익 시간축과 실험 전략 판단

## 필터링 기준

| 기준 | 질문 |
| --- | --- |
| 최신성 | 이 정보는 지금도 유효한가? |
| 반응성 | 사람들이 실제로 관심을 보였는가? |
| 신뢰성 | 출처가 믿을 만한가? |
| 사업 적합성 | 우리 제품, 타깃, 가격, 채널에 맞는가? |
| 시간축 적합성 | 단기 수익, 중기 검증, 장기 자산 중 어디에 맞는가? |

## 적용 규칙

- 기발하지만 반응 근거가 없는 정보는 아이디어 후보로만 둔다.
- 오래된 트렌드 자료는 현재 플랫폼/고객 반응과 다시 대조한다.
- 오래된 학술 이론은 기본 원리로 사용할 수 있지만, 시장 실행안은 최신 자료로 보정한다.
- 장기 브랜드 전략에서도 초반 유입이 필요하면 브랜드 가치를 해치지 않는 트렌드를 벤치마킹한다.
