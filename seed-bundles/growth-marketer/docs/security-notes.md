# Security Notes

- 고객 이메일, 전화번호, 결제 정보, CRM 데이터는 민감정보를 제거한 뒤 입력한다.
- 광고 계정, 분석 계정, CRM 계정 권한은 v1에서 요구하지 않는다.
- 외부 모델 API를 사용할 경우 입력 내용과 메모리 파일 내용이 API 제공자에게 전송될 수 있다.
- `memory/` 파일에는 API 키, 비밀번호, 고객 개인정보, 비공개 계약 조건을 저장하지 않는다.
- 의료, 법률, 금융, 투자, 채용, 부동산 등 규제 산업의 마케팅 문구는 전문가 검토가 필요하다.
- 허위 후기, 허위 수량 제한, 기만적 할인, 과장 효능 표현은 만들지 않는다.
