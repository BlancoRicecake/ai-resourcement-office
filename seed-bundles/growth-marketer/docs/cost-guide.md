# Cost Guide

이 번들은 기본적으로 프롬프트형 직원 패키지다.

예상 비용:

- Claude, Codex, ChatGPT 등 사용 중인 플랫폼 요금제에 따른다.
- API로 직접 실행할 경우 입력 길이, 메모리 파일 길이, 설계서 분량에 따라 모델 사용료가 발생한다.
- 광고 집행, 이메일 발송 도구, 분석 도구, CRM 비용은 번들에 포함되지 않는다.

권장:

- 처음에는 광고비 없이 콘텐츠, 랜딩페이지 문구, 이메일 아웃리치 실험부터 테스트한다.
- 유료 광고는 목표, 예산, 중단 기준을 확정한 뒤 집행한다.
- 메모리 파일이 길어지면 핵심만 압축해 유지한다.
- 대량 캠페인 생성 전 한 가지 목표와 한 가지 채널로 출력 품질을 확인한다.
