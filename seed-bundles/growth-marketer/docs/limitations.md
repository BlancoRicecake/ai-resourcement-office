# Limitations

- 실제 광고 집행, 계정 연결, 예산 사용은 v1 범위에 포함하지 않는다.
- 최신 광고 플랫폼 정책, 개인정보 규제, 광고 단가, 알고리즘 변화는 별도 조사가 필요하다.
- 데이터가 부족하면 퍼널 진단과 수익 시간축 판단은 가정 기반으로 작성된다.
- 실험안은 의사결정 참고자료이며 실제 매출 증가를 보장하지 않는다.
- 법적 검토가 필요한 광고 문구나 업종별 심의 문구는 사용자가 최종 검토해야 한다.
- 메모리 파일은 모델 자체 학습이 아니며, 다음 세션에서 읽어 적용하는 운영 기록이다.
- 잘못된 판단이 메모리에 들어가면 이후 출력에 영향을 줄 수 있으므로 사용자 승인 후 반영해야 한다.
