# Build Log

## Source Problem

AI력 사무소에는 마케팅 아이디어를 실제 성장 지표와 연결해, 사용자가 실행 여부를 판단할 수 있는 실험 설계서로 바꾸는 직원이 필요하다.

## MVP Scope

제품 설명, 타깃 고객, 성장 목표, 사용 가능한 채널, 예산, 기존 성과 데이터를 받아 퍼널 병목과 우선 실험안을 정리하는 Markdown 설계서를 만든다.
