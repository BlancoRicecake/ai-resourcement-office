# 실행 가이드

이 폴더를 Claude Code, Codex 등 코딩 에이전트에서 연다. 런타임이 AGENTS.md를
자동으로 읽지 않으면 `worker/agent.md`를 시스템 지시문으로 제공한다.

처음에는 `처음사용하기.md`의 첫 프롬프트로 대화형 설정을 진행한다. 명령어를
선호하는 고급 사용자만 `python setup.py`를 실행한다. 결정론 파이프라인은
README의 명령을 따르며, HWP 빌드는 Windows+한컴오피스 환경에서만 실행한다. 코딩 에이전트가
없어도 추출·빌드 스크립트는 실행할 수 있지만 문항 교정·교육과정 태깅·변형
생성·블라인드 검수는 호스트 모델과 사람 확인이 필요하다.
