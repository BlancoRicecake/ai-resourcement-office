# 변형문제 품질 규칙

변형 전 문항의 alignment가 confirmed이고 review_required=false여야 한다.
VariantSpec에는 교육과정, 단원, 성취기준, 출제 의도 불변조건, 변경 요소,
난이도 추론 단계, 정답 증명, 블라인드 독립 풀이, 적용 검사, 그림 계약을 담는다.

검수자는 생성 정답과 change_note를 보지 않고 다시 푼다. 정답 일치만으로 통과
처리하지 않고 풀이 근거와 정답 유일성을 기록한다. 그림을 재사용하면 발문·수치·
라벨 충돌 여부를 별도로 확인한다. qa.status가 pass가 아니면 출력 게이트를
통과시키지 않는다.
