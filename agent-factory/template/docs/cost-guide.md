# Cost Guide

결정론 도구(`app/core`, CLI, MCP)는 로컬에서 돌아가며 **$0**입니다. 비용이
드는 것은 판단(진단·리라이트 등)을 수행하는 **호스트 코딩 에이전트/LLM의
토큰**뿐입니다 — 이 번들은 자체 API 키를 요구하지 않습니다
(`agent-factory/PRINCIPLES.md` 원칙 1).
