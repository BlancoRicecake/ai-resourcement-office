발견하셨다면, **공개 이슈로 올리지 말고** 아래로 비공개 제보해 주세요.

- 이메일: **heobusy@gmail.com**
- 제목에 `[SECURITY]`를 붙여 주시면 빠르게 확인합니다.

가능하면 다음을 함께 알려주세요.

- 문제가 있는 파일 / 페이지 / 다운로드 패키지
- 재현 방법 또는 예시
- 예상되는 영향

책임 있는 제보에 감사드리며, 확인 후 회신드립니다.

## 이 프로젝트의 보안 모델

AI력 사무소는 **실행형 호스팅 서비스가 아닙니다.** 재사용 가능한 패키지를
배포할 뿐이며, 다음은 전적으로 사용자 책임입니다.

- API 키 (사용자 본인의 키를 사용자 환경에서만 사용)
- 모델 사용 비용 / 클라우드 비용
- MCP 권한, 데이터 보안, 결과 검증

배포되는 파이썬 앱은 외부 서드파티 라이브러리를 사용하지 않고 파이썬 표준
라이브러리만 사용합니다. API 키는 코드에 저장되지 않으며, 사용자가 지정한
LLM 엔드포인트로만 전송됩니다.

## 다운로드 무결성 검증

배포되는 팀 패키지(zip)의 SHA-256 체크섬은 [`CHECKSUMS.txt`](./CHECKSUMS.txt)에
공개되어 있습니다. 다운로드한 파일이 변조되지 않았는지 다음처럼 확인할 수 있습니다.

```bash
# macOS / Linux
shasum -a 256 다운로드한-파일.zip

# Windows (PowerShell)
Get-FileHash 다운로드한-파일.zip -Algorithm SHA256
```

출력된 값이 `CHECKSUMS.txt`의 값과 같으면 원본과 동일한 파일입니다.
